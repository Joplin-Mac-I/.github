## Overview

Joplin Advanced is an open-source markdown note-taking application that stores notebooks of encrypted notes with flexible sync options, including self-hosted servers and standard third-party cloud storage.

End-to-end encryption protects note content before it ever leaves the device, ensuring that even the sync service transporting your notes cannot read their content — a guarantee closed-source note apps cannot make.

Sync target flexibility supports Nextcloud, WebDAV, Dropbox, OneDrive, and other services, letting you choose exactly where your encrypted notes are stored rather than being locked into a single provider.

A command-line interface and REST API expose the same notebook data to scripts and external tools, making Joplin practical to integrate into developer workflows beyond the graphical application.

Note history tracking preserves previous versions of every note automatically, providing a built-in undo mechanism that extends beyond a single editing session.

Custom CSS theming allows the rendered markdown preview to be styled to match personal preference, giving Joplin Advanced the same visual flexibility as the open formats it stores notes in.

For privacy-conscious users who want full transparency over their note-taking tool and control over where their data physically lives, Joplin Advanced is the open-source notebook built around both of those guarantees.

## Performance

Joplin balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
